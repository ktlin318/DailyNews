# 事件整併與焦點評分規則

## 同一事件判斷

四項皆實質相符才可合併：

1. 行動者相同，例如同一主管機關、公司或人物。
2. 核心行動相同，例如公布、核准、調整、發布或發生。
3. 主體相同，例如同一政策、財報期、產品或事故。
4. 事件時間相同或屬同一次發布。

只有公司名稱或產業相同，不足以合併。

## 應分開的情況

- 傳聞與正式公告。
- 政策草案與正式定案。
- 公司決議與後續執行結果。
- 事件本身與隔日市場反應。
- 同一公司不同產品、不同地區或不同財報期。

## 來源獨立性

- 原始公告、主管機關資料及獨立媒體可各視為不同證據。
- 同一通訊社稿件的轉載頁只算一個獨立報導來源，但網址仍可保留供使用者開啟。
- 網頁名稱必須和網址實際所屬網站一致。

## 評分證據

- 重要性：影響範圍、政策層級、金額規模、是否改變既有狀態。
- 跨來源覆蓋：獨立來源數與來源品質。
- 首頁或熱門：實際首頁、熱門榜、頭條或明確版位。
- 客群相關性：是否能支持目前客群的合理對話，而非銷售機會想像。
- 新鮮度：相對於搜尋窗口截止時間的發布時間。

不得以語氣聳動、品牌知名度或行銷文案代替證據。
